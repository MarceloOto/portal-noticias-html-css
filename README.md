# Portal de Notícias HTML/CSS

Projeto acadêmico desenvolvido com HTML e CSS, inspirado no layout de portais de notícias para fins de estudo.

## Objetivo

Praticar conceitos de desenvolvimento web utilizando:

* HTML5
* CSS3
* Estruturação de páginas
* Responsividade
* Organização de conteúdo

## Funcionalidades

* Página inicial de portal de notícias
* Layout organizado em seções
* Estilização com CSS
* Estrutura responsiva

## Tecnologias Utilizadas

* HTML5
* CSS3

## Como executar o projeto

1. Faça o download do projeto.
2. Abra o arquivo `index.html` em qualquer navegador.
